# Variační Koeficienty

## `VARCOEF`

Populační variační koeficient.

### Syntaxe

```excel
=VARCOEF(values)
```

### Výstup

Skalární `σ / μ`.

## `VARCOEF.S`

Výběrový variační koeficient.

### Syntaxe

```excel
=VARCOEF.S(values)
```

### Výstup

Skalární `sₓ / x̄`.

## `VARCOEF.S.W`

Vážený výběrový variační koeficient.

### Syntaxe

```excel
=VARCOEF.S.W(values; weights)
```

### Výstup

Skalární `sₓ,w / x̄_w`.
